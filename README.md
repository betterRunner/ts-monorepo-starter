# ts-monorepo-starter

## Introduction

This project is a starter to build your own monorepo library with typescript. 

Details:

- managed by [pnpm](https://pnpm.io/)
- built by [rollup.js](https://rollupjs.org)
- tested by [jest](https://jestjs.io)

## Getting Started

### Install

```bash
pnpm i
```

### Test
```bash
pnpm run test
```

### Build
```bash
pnpm run build
```
